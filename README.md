Build a HelloWorld on Solana from scratch

- An on-chain Hello World Program.
- A client can send a "Hello" to an account and get back the number of times "hello" has been sent.


## Client

- @solana/web3.js

## Usage

```
make build
make deploy
make run-client
```

> See [Makefile](./Makefile)
